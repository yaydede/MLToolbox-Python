# Machine Learning Toolbox — Python Companion

Python translations of the code chapters from **Yigit Aydede,
*Machine Learning Toolbox for Social Scientists*** (Taylor & Francis, 2023),
developed as part of the AtlanticOER open-textbook project
*Introduction to Applied Machine Learning with R and Python for Social Sciences and Business*.

Each file mirrors a chapter of the original R material, re-implemented in Python so that
every method is available in both languages.
